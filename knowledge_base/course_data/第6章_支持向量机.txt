## 第 6 章  支持向量机

### 6.1 间隔与支持向量
二分类任务中，线性划分超平面通用表达式：w 转置・x + b = 0。向量 w 是超平面法向量，控制超平面朝向；标量 b 是偏移量，控制超平面到原点距离。样本点 x 到超平面几何距离公式：r = |w 转置・x + b| / ||w||，||w|| 代表 w 的 L2 范数。
对于线性可分数据集，标签 y 取值 + 1、-1，完美划分超平面满足约束：若 yi=+1，则 w 转置・xi + b ≥ 1；若 yi=-1，则 w 转置・xi + b ≤ -1。满足等号条件的样本称为支持向量。异类支持向量到超平面距离相加称为间隔，间隔大小：γ = 2 / ||w||。
最大间隔优化目标：最小化 (1/2)・||w|| 平方，约束全部样本满足 yi・(w 转置・xi + b) ≥ 1。最小化 ||w|| 等价最大化间隔，该最优超平面唯一，相比感知机随机分割面，鲁棒性、泛化性能更强，天然贴合结构风险最小化理论。仅依靠少量支持向量即可确定分割边界，无关样本不影响最终超平面。

### 6.2 对偶问题
原始最大间隔优化是带不等式约束的凸二次规划，直接求解开销大，引入拉格朗日乘子 αi ≥ 0 构造拉格朗日函数：
L (w,b,α) = (1/2)||w|| 平方 + 求和_{i=1 到 m} αi・[1 - yi・(w 转置・xi + b)]
分别对 w、b 求偏导并置零，得到两个等式：
w = 求和_{i=1 到 m} αi・yi・xi
求和_{i=1 到 m} αi・yi = 0
将两式代回原拉格朗日函数，消去 w、b，得到对偶优化问题：
最大化 求和_{i=1 到 m}αi - (1/2) 求和_{i=1 到 m} 求和_{j=1 到 m} αiαj yi yj (xi 转置 xj)
约束条件：αi≥0，求和 αi yi = 0
根据 KK 互补条件，仅当样本为支持向量时 αi > 0，非支持向量对应的 αi 全部等于 0，最终模型仅依赖支持向量计算预测值。
对偶形式优势：训练仅需样本内积运算，适配核技巧；经典求解算法 SMO（序列最小优化）每次仅更新两个拉格朗日乘子，解析求解子问题，迭代速度远优于通用二次规划工具。

### 6.3 核函数
硬间隔 SVM 仅能处理线性可分数据，现实多数任务线性不可分，解决思路：通过映射函数 φ 将原始低维样本映射至高维希尔特征空间，使高维空间数据线性可分。
直接显式高维映射会引发维度灾难，计算内积开销爆炸，核技巧规避该问题：定义核函数 K (xi,xj) = φ(xi) 转置・φ(xj)，无需构造 φ，直接在原空间计算高维内积。
替换对偶问题内积项 xi 转置 xj 为 K (xi,xj)，得到非线性 SVM 对偶目标：
最大化 求和 αi - (1/2) 求和求和 αiαj yi yj K (xi,xj)
常用标准核函数：
线性核：K (xi,xj)=xi 转置 xj，等价原始线性 SVM；
多项式核：K (xi,xj)=(xi 转置 xj + p)^d，d 为多项式次数，p 常数；
高斯核（RBF 径向基核）：K (xi,xj)=exp (-||xi-xj|| 平方 /(2σ 平方))，σ 控制核宽度，适配各类非线性分布，使用最广泛；
Sigmoid 核：K (xi,xj)=tanh (ρ xi 转置 xj + c)。
核函数有效性判定：核矩阵半正定；核函数可线性组合、乘积生成新合法核。

### 6.4 软间隔与正则化
硬间隔 SVM 强制所有样本正确划分，对噪声、异常点极度敏感，微小离群样本即可破坏最大间隔超平面。引入软间隔机制，允许少量样本违反间隔约束，增加松弛变量 ξi ≥ 0。
软间隔原始优化目标：最小化 (1/2)||w|| 平方 + C・求和_{i=1 到 m} ξi
约束条件：yi (w 转置 xi + b) ≥ 1 - ξi，ξi ≥ 0
常数 C 为惩罚系数：C 越大，对分错样本惩罚越强，越趋近硬间隔；C 越小允许更多样本越过间隔边界。
构造拉格朗日函数推导对偶问题，KK 条件更新为 0 ≤ αi ≤ C，此时 αi=0：样本正确且间隔充足；0<αi<C：支持向量、落在间隔边界；αi=C：样本违反间隔、被分错或落在间隔内部。
从正则化视角理解软间隔：目标第一项是模型复杂度惩罚，第二项是训练损失，C 平衡模型复杂度与训练误差，契合结构风险最小化，缓解过拟合。

### 6.5 支持向量回归 (SVR)
传统回归最小化预测与真实值全局误差，SVR 引入 ε 不敏感间隔带：预测值与真实标签差值绝对值小于 ε 不计损失，仅超出区间样本产生回归损失。
引入上下两组松弛变量 ξi 帽、ξi 星，原始优化目标：最小化 (1/2)||w|| 平方 + C・求和 (ξi 帽 + ξi 星)
约束：yi - (w 转置 xi + b) ≤ ε + ξi 帽；(w 转置 xi + b) - yi ≤ ε + ξi；ξi 帽≥0，ξi 星≥0
拉格朗日对偶推导后同样仅保留支持向量参与预测，引入核函数可实现非线性回归，称为核 SVR，适配非线性拟合任务。

### 6.6 核方法
核技巧并非仅用于 SVM，是通用算法改造框架：任何仅依赖样本内积的线性学习器，均可将内积替换为核函数，拓展至非线性场景，整套体系称为核方法。
经典核化拓展模型：核 PCA、核 LDA、核岭回归、核聚类等。核方法核心优势：不修改原始算法逻辑，仅替换内积运算，低复杂度实现非线性建模；局限：样本量巨大时核矩阵存储、计算成本极高。

### 6.8 休息一会儿 小故事

支持向量机理论奠基人弗拉基米尔・瓦普尼克（Vladimir Vapnik），上世纪六七十年代提出 VC 维、统计学习理论，但受限于当年算力，理论长期无法落地；九十年代高效 SVM 求解算法出现后，SVM 迅速席卷模式识别、文本分类领域，取代传统神经网络成为主流模型。瓦普尼克长期任职贝尔实验室，统计学习理论奠定了小样本机器学习的理论根基。
