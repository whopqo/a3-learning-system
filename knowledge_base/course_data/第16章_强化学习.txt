## 第 16 章  强化学习

### 16.1 任务与奖赏

### 16.1.1 强化学习基础定义
强化学习（RL）核心目标：智能体（Agent）与环境持续交互，通过试错学习状态到动作的映射策略，最大化长期累积奖赏。
和监督 / 无监督学习核心区分：
监督学习：每一条样本拥有明确标准答案标签，一次性给出完整监督信号；
无监督学习：无任何反馈信号，仅挖掘数据内在结构；
强化学习：不存在即时标准答案，仅执行动作后环境给出单步奖赏（可正可负），需要延迟回报、序列决策优化。

### 16.1.2 MDP 马尔可夫决策过程
全部强化学习任务统一建模为马尔可夫决策过程 MDP，五元组构成：
状态集合 S：环境所有可观测情况；
动作集合 A：智能体可执行全部操作；
状态转移概率 P (s' | s,a)：在状态 s 执行动作 a，转移到 s' 的概率，满足马尔可夫性：下一状态仅依赖当前 s、a，与历史无关；
即时奖赏函数 r (s,a,s')：执行动作 a 从 s 到 s' 获得即时奖励；
折扣因子 γ ∈ [0,1]：远期奖赏衰减系数，γ=1 无限远期同等重要，γ 趋近 0 只看重当前一步收益。

### 16.1.3 策略、价值函数
策略 π(a|s)：给定状态 s，选取动作 a 的概率分布；确定性策略 π(s) 直接输出单一最优动作。
状态价值 V^π(s)：从状态 s 出发，遵循策略 π 得到的长期折扣累积奖赏期望
V^π(s) = E [ r1 + γr2 + γ²r3 + ... | s0=s, π ]
动作价值 Q^π(s,a)：状态 s 执行动作 a，再遵循策略 π 的长期累积奖赏期望
Q^π(s,a) = E [ r1 + γV^π(s1) | s0=s,a0=a ]
二者关系：V^π(s) = 求和_{a} π(a|s) Q^π(s,a)

### 16.1.4 贝尔期望方程
状态价值贝尔方程：
V^π(s) = 求和_{a} π(a|s) 求和_{s'} P (s'|s,a) [ r (s,a,s') + γ V^π(s') ]
最优价值 V*(s) = max_a Q*(s,a)，最优动作价值满足最优贝尔方程：
Q*(s,a) = 求和_{s'} P (s'|s,a) [ r (s,a,s') + γ max_{a'} Q*(s',a') ]

### 16.1.5 回合式与持续任务
回合任务：存在终止状态，每一轮交互有结束点（下棋、游戏通关），累积奖赏求和到终止；
持续任务：无终止状态（机器人持续行走），依靠折扣因子 γ 避免奖赏求和无穷大。

### 16.2 K - 摇臂赌博机（单状态强化学习）

### 16.2.1 探索与利用困境
仅单一状态、K 个可选动作，无状态转移，核心矛盾：
利用（Exploit）：选择当前已知平均奖赏最高的动作，最大化短期收益；
探索（Explore）：尝试低奖赏动作，挖掘潜在更高回报，牺牲短期换取长期最优；
完全利用易陷入局部最优，纯探索收益极低，需要平衡二者。

### 16.2.2 ε- 贪心算法
全局超参数 ε∈(0,1)：
1-ε 概率贪心选择当前平均奖赏最大动作（利用）；
ε 概率均匀随机选取任意动作（探索）；
ε 越大探索越强，越小越保守贪心。
记录每个动作 a 的执行次数 count (a)、累计总奖赏 sum_r (a)，平均奖赏 Q (a)=sum_r (a)/count (a)。

### 16.2.3 Softmax（玻尔兹曼探索）
基于概率分布平衡探索，动作选择概率：
π(a) = exp (Q (a)/τ) / 求和_{a'} exp (Q (a')/τ)
τ 为温度系数：τ 大所有动作概率接近，探索强；τ 趋近 0 退化为贪心策略。

### 16.2.4 上置信界 UCB
不再随机探索，为每个动作构造置信上界，选取 UCB 最大动作：
UCB (a) = Q (a) + sqrt ( ln (t) /count (a) )
t 为总执行次数，执行越少的动作置信项越大，自动引导探索稀缺动作，无随机波动，确定性探索策略。

### 16.3 有模型学习

### 16.3.1 模型已知定义
智能体完全掌握环境转移概率 P (s'|s,a) 与奖赏 r (s,a,s')，无需交互采样估计环境，可直接通过贝尔方程迭代求解最优策略，代表算法：策略迭代、值迭代。

### 16.3.2 策略评估
输入固定策略 π，迭代更新全部状态价值直至收敛，流程：
初始化所有状态 V (s)=0；
遍历每个状态 s，用贝尔期望方程更新 V_new (s)；
计算本轮最大更新差值 Δ；
Δ 小于预设阈值 θ 则收敛停止；
输出当前策略对应的完整状态价值表。

### 16.3.3 策略改进
基于当前价值函数贪心更新策略：对每个状态 s，新策略选取使 Q (s,a) 最大的动作，生成更优确定性策略 π'；
理论保证：改进后策略累积奖赏一定不低于原策略，单调提升。

### 16.3.4 策略迭代算法
完整两阶段循环：
策略评估：固定 π 计算全部状态价值；
策略改进：贪心生成新策略；
循环直到策略不再发生变化，得到全局最优策略。
优势：收敛稳定，每轮策略严格优化；缺陷每轮评估迭代开销大。

### 16.3.5 值迭代算法
合并评估与改进，直接迭代最优贝尔方程，不保存完整中间策略：
初始化 V (s)=0；
对每个 s 更新 V_new (s) = max_a 求和_{s'} P (s'|s,a)[r+γV (s')]；
差值小于阈值停止；
收敛后贪心提取最优策略。
优势单次迭代速度快，工业更常用；缺陷丢失中间策略信息。

### 16.4 免模型学习
现实场景环境转移概率 P 未知，无法构建 MDP 模型，只能通过与环境交互采样轨迹，分为蒙特卡罗、时序差分 TD 两大类。

### 16.4.1 蒙特卡罗强化学习
核心：完整跑完一整个回合，利用回合结束后的完整真实累积奖赏更新价值，无单步自举。
同策略 MC：使用 ε- 贪心生成轨迹，更新同策略价值；
异策略 MC：重要性采样，用探索策略采集轨迹，评估贪心最优策略，消除探索动作带来偏差；
更新公式：每访问一次状态 s，取整条轨迹真实折扣回报 G，滑动平均更新 V (s)；
优点：无自举偏差；缺点必须等回合结束，长序列效率极低。

### 16.4.2 时序差分 TD 学习
单步更新，不需要等待回合终止，利用下一时刻预估价值自举（bootstrap），是免模型主流。
TD (0) 单步时序差分
更新目标：TD 目标 = r + γV (s')
更新量：V (s) = V (s) + α [ TD 目标 - V (s) ]，α 为学习率；
SARSA（同策略 TD）
更新动作价值 Q，轨迹动作与更新策略完全一致，名称来自五元组 (s,a,r,s',a')；
Q-learning（异策略 TD，最经典）
行为策略 ε- 贪心探索，更新时直接取下一状态最大 Q 值，分离探索与最优价值估计；
更新式：Q (s,a) ← Q (s,a) + α[ r + γ max_{a'} Q (s',a') - Q (s,a) ]
优势：直接收敛到最优 Q 值，无需等待完整回合，游戏、机器人广泛使用。

### 16.4.3 n 步 TD
平衡 MC 与 TD (0)，取连续 n 步真实奖赏加 n 步后预估价值，n=1 等价 TD (0)，n 等于回合长度等价蒙特卡罗。

### 16.5 值函数近似

### 16.5.1 查表法局限
离散小规模 S/A 可用表格存储 V/Q；高维连续状态（图像、传感器数据）状态空间无穷大，表格无法存储，必须使用参数化近似函数。
近似形式：V_w (s)、Q_w (s,a)，w 为模型参数（线性模型、神经网络）。

### 16.5.2 线性近似
特征向量 φ(s)，V_w (s)=w 转置・φ(s)，损失为 TD 误差平方，梯度下降更新权重，理论稳定易收敛。

### 16.5.3 深度强化学习（非线性近似）
用神经网络拟合 Q 网络，即 DQN，两大关键改进解决训练震荡：
经验回放池：存储历史交互样本，随机批量采样打破样本时序相关性；
目标网络：分离实时网络与延迟目标网络，稳定 TD 目标；
缺陷非线性近似会破坏理论收敛保证，易发散。

### 16.5.4 策略梯度近似
不拟合价值，直接参数化策略 π_θ(a|s)，沿累积奖赏梯度上升更新 θ，代表算法 REINFORCE、Actor-Critic：
Actor：优化策略；Critic：拟合价值基线降低梯度方差。

### 16.6 模仿学习
纯强化学习依赖稀疏奖赏，大量复杂任务难以设计有效奖赏函数，模仿学习利用专家示范数据降低学习难度。

### 16.6.1 直接模仿学习（行为克隆）
监督学习思路：把专家轨迹 (s,a) 作为监督样本，拟合状态到动作映射；
缺陷：分布偏移，智能体偏离专家状态后无修正信号，误差持续累积。

### 16.6.2 逆强化学习 IR
不直接复制动作，反向推导专家行为背后的奖赏函数：
假设专家策略最优；
求解奖赏 r，使专家轨迹累积奖赏显著高于随机策略；
得到奖赏后再运行标准 RL 训练智能体；
解决人工设计奖赏困难，自动驾驶、机器人操控常用。

### 16.8 休息一会儿 小故事

强化学习思想最早可追溯到心理学动物试错实验，1950s 萨缪尔跳棋程序是最早 RL 落地系统。Q-learning 由 Watkins 提出，彻底简化免模型最优策略求解；DQN 在 Atari 游戏取得突破，开启深度强化学习浪潮。逆强化学习解决奖赏工程痛点，如今机器人、自动驾驶大量采用模仿 + RL 结合框架。
