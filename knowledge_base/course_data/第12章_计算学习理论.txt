## 第 12 章  计算学习理论

### 12.1 基础知识
计算学习理论（CLT）是机器学习的理论根基，不聚焦单一算法实现，而是从数学层面回答三大核心问题：1）什么样的学习任务是可解的；2）完成学习需要多少样本（样本复杂度）；3）学习器泛化误差存在怎样的理论上界，为模型设计、模型选择提供严格数学保障。

### 12.1.1 基础符号与误差定义
样本空间 X，标记空间 Y={0,1}（二分类简化讨论），未知独立同分布 D；
假设 h：X→Y，映射样本到类别；全部假设构成假设空间 H；
真实目标概念 c：样本真实分类规则，c∈H；
泛化误差 R (h)：h 在全局分布 D 上的错误概率
R (h) = E_{x~D} [ I (h (x)≠c (x)) ]
经验误差 R 帽 (h)：h 在训练集 D_m（m 个 i.i.d 样本）上的错误率
R 帽 (h) = (1/m) * 求和_{i=1}^m I (h (x_i)≠c (x_i))
泛化误差是真实性能，经验误差是训练集观测值，二者存在差距，理论核心就是量化二者差值的上界。

### 12.1.2 核心不等式（泛化界数学工具）
Hoeffding 不等式
设 m 个独立 0-1 随机变量，任意 ε>0，
P (|R 帽 (h) - R (h)| ≥ ε ) ≤ 2 * exp ( -2mε² )
含义：单条假设下，经验误差偏离真实误差超过 ε 的概率随样本量指数衰减。
并界（联合界）
若假设空间 H 包含 M 个假设，则至少一个假设满足 |R 帽 (h)-R (h)|≥ε 的概率：
P (存在 h∈H, |R 帽 (h)-R (h)|≥ε ) ≤ M * 2*exp (-2mε²)
有限假设空间核心推导基础。
McDiarmid 不等式
有界函数作用于独立变量，函数值偏离期望的概率指数衰减，用于 Rademacher 复杂度推导。

### 12.2 PAC 学习（概率近似正确）

### 12.2.1 PAC 形式化定义
PAC 是 “成功学习” 的标准数学定义，包含两个参数：
近似参数 ε>0：允许泛化误差上限，代表模型近似程度；
置信参数 δ∈(0,1)：允许失败概率上限。
给定概念类 C，若存在学习算法 L，满足：
任意目标概念 c∈C、任意分布 D、任意 ε,δ>0，
存在样本复杂度 m (ε,δ)，当训练样本数 m≥m (ε,δ) 时，
算法输出假设 h 满足 P (R (h) ≤ ε) ≥ 1-δ，
则称概念类 C 是 PAC 可学习的。
通俗解释：取足够多样本后，有至少 1-δ 的概率得到误差不超过 ε 的模型。

### 12.2.2 样本复杂度
满足 PAC 条件所需最小样本数量 m (ε,δ)，反映学习任务的数据需求，是 PAC 理论核心输出。

### 12.2.3 可分与不可分区分
可分场景：假设空间中存在 h 使得经验误差 = 0（训练集完全正确划分）；
不可分场景：H 内不存在零误差假设，只能最小化经验损失。

### 12.3 有限假设空间
假设空间 H 大小 | M | 有限，仅使用 Hoeffding + 并界即可推导出泛化界。

### 12.3.1 可分情形（存在零训练误差假设）
学习算法输出训练集无错的假设 h；
推导泛化误差上界：
固定 ε,δ，令 M2exp (-2mε²) ≤ δ，解出：
m ≥ (1/(2ε²)) * ln (2M/δ)
样本复杂度 m (ε,δ) = O ( (1/ε²) ln (M/δ) )
结论：假设空间越大、要求精度越高、置信度越高，所需样本量越大。

### 12.2 不可分情形（无零误差假设）
算法最小化经验误差 min_h R 帽 (h)；
并界给出统一泛化约束：以至少 1-δ 概率，对全部 h∈H：
R (h) ≤ R 帽 (h) + sqrt ( ln (2M/δ) / (2m) )
最优假设 h 星的泛化误差不会超过训练误差加该项复杂度惩罚；
惩罚项随假设空间大小对数增长，随样本量平方根递减。

### 12.4 VC 维（Vapnik-Chervonenkis 维）
有限假设空间仅适用于离散小模型，SVM、神经网络等无限假设空间无法用 M 度量，VC 维衡量无限 H 的表达能力，是统计学习理论核心指标。

### 12.4.1 打散 Shatter
给定样本集合 Z（大小 d），若假设空间 H 能实现 Z 上全部 2^d 种 0/1 标记划分，则称 H 可以打散 Z。

### 12.4.2 VC 维定义
VC (H) = H 能够打散的最大样本数目；
若可打散任意规模样本，VC (H)= 无穷大（这类任务不可 PAC 学习）。

### 12.4.3 增长函数 Π_H (m)
m 个样本最多能产生多少种不同划分，Π_H (m) = max_{|Z|=m} |{h 在 Z 上的划分，h∈H}|
增长函数上界：Π_H (m) ≤ m^{VC (H)} + 1，刻画假设空间随样本扩张速度。

### 12.4.4 无限假设空间泛化界
将有限空间的 M 替换为增长函数，结合 VC 维得到通用泛化上界：
以 1-δ 概率，对任意 h∈H：
R (h) ≤ R 帽 (h) + sqrt ( 2 ln Π_H (m)/m ) + sqrt ( ln (1/δ)/(2m) )
样本复杂度由 VC 维主导：m=O ( VC (H)/ε² )

### 12.4.5 常见模型 VC 维
d 维线性分类器（超平面）：VC=d+1；
单隐层神经网络（k 个隐节点）：VC 与隐节点数量线性相关；
决策树（深度 t）：VC=O (t log t)；
高斯核 SVM：VC 无穷，但核方法存在隐正则，实际可学习。

### 12.5 Rademacher 复杂度
VC 维仅给出最坏上界，Rademacher 复杂度通过数据分布自适应度量假设空间复杂度，界更紧致。

### 12.5.1 Rademacher 随机变量
σ_i 独立取 + 1、-1，概率各 0.5；

### 12.5.2 经验 Rademacher 复杂度
R_m (H) = E_σ [ sup_{h∈H} (1/m) 求和_{i=1}^m σ_i h (x_i) ]
直观：随机翻转样本标签，假设空间能拟合随机噪声的程度，拟合能力越强复杂度越高。

### 12.5.3 泛化误差界
以 1-δ 概率：
R (h) ≤ R 帽 (h) + R_m (H) + sqrt ( ln (1/δ)/(2m) )
优势：不依赖最坏打散情况，适配真实数据分布，VC 维是与分布无关的最坏界，Rademacher 是数据依赖的紧致界。

### 12.5.4 与增长函数关系
R_m (H) ≤ sqrt ( 2 ln Π_H (m) /m )，建立二者数学联系。

### 12.6 稳定性
VC、Rademacher 从假设空间复杂度出发，稳定性从算法自身性质推导泛化保障，不依赖假设空间大小。

### 12.6.1 均匀稳定定义
学习算法 L，移除任意单个样本后输出假设的损失变化不超过 β，则称算法 β 均匀稳定，β 为稳定系数，β 越小越稳定。

### 12.6.2 稳定算法泛化界
均匀稳定算法存在不依赖 VC 维的泛化上界，仅由稳定系数、样本量、置信度决定。

### 12.6.3 算法稳定性对比
稳定算法：岭回归、SVM（带正则），删除单个样本预测变化极小；
不稳定算法：无剪枝深决策树、k=1 近邻，少量样本变动预测剧烈；

### 12.6.4 核心结论
正则化是提升算法稳定性的核心手段，正则项约束模型复杂度，降低对单样本的依赖。

### 12.8 休息一会儿 小故事

VC 维两位提出者 Vapnik 与 Chervonenkis 于 1968 年发表相关理论，但早年算力不足，理论长期无法落地；90 年代 SVM 出现后 VC 维成为支撑统计学习的核心理论。Valiant 在 1984 年提出 PAC 框架，奠定计算学习理论完整体系，2010 年凭借 PAC 理论获图灵奖。整套理论解释了 “为什么复杂模型容易过拟合”，为正则化、模型简化提供严格数学依据。
