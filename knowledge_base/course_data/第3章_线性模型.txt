## 第 3 章  线性模型

### 3.1 基本形式
给定由 d 个属性描述的样本向量 x，x 写作列向量 x=(x1;x2;…;xd)，线性模型通用表达式：
f (x) = w1x1 + w2x2 + … + wd*xb
向量简写形式：f (x) = w 转置・x + b
其中 w=(w1;w2;…;wd) 是权重向量，b 为偏置项。
权重 w 可直观体现各属性对预测结果的重要程度，线性模型可解释性强。大量非线性模型依托线性模型，通过特征映射、层级结构拓展得到。

### 3.2 线性回归

### 3.2.1 单变量线性回归
数据集 D={(xi,yi)}，xi 为单输入属性，目标拟合直线 f (xi)=wxi+b，尽可能让 f (xi) 接近真实 yi。
采用均方误差作为损失，最小二乘法求解参数 w、b：
目标最小化 求和从 i=1 到 m (yi - wxi - b) 的平方
对 w、b 分别求导并令导数等于 0，得到闭式解：
w = 求和 (yi*(xi - x 均值)) / 求和 (xi 平方 - (1/m)*(求和 xi) 平方)
b = (1/m)求和 (yi - wxi)
x 均值代表全部样本属性平均值。

### 3.2.2 多元线性回归
样本含 d 个属性，构造增广矩阵 X，每行末尾补充常数 1，增广权重向量 w 帽 =(w;b)，标签向量 y=(y1;y2;…;ym)。
损失函数：(y - X・w 帽) 的转置 点乘 (y - X・w 帽)
对 w 帽求导置零，最优解：w 帽星号 = (X 转置・X) 的逆・X 转置・y
若 X 转置・X 不满秩，存在多组最优解，依靠正则化、归纳偏好选择最终参数。

### 3.3 广义线性模型
普通线性回归：y = w 转置 x + b
对数线性回归：ln (y) = w 转置 x + b，等价 y = exp (w 转置 x + b)
通用广义线性模型：y = g 逆 (w 转置 x + b)
g 称为联系函数，g 单调可微。

### 3.3 对数几率回归（逻辑回归）
二分类任务标签 y 取值 0 或 1，阶跃函数不连续无法直接优化，替换为 Sigmoid 对数几率函数：
y = 1 / (1 + exp (-z))，其中 z=w 转置 x+b
变形得到对数几率形式：ln (y/(1-y)) = w 转置 x + b
y 代表样本属于正类的后验概率，推导得：
p (y=1|x) = exp (w 转置 x+b) / (1+exp (w 转置 x+b))
p (y=0|x) = 1 / (1+exp (w 转置 x+b))
采用极大似然估计构造损失函数，令 β=(w;b)，x 帽 =(x;1)，最大化对数似然等价最小化：
求和从 i=1 到 m [ -yi・β 转置 x 帽 i + ln (1+exp (β 转置 x 帽 i)) ]
该损失函数为凸函数，可用梯度下降、牛顿法求解最优 β。牛顿法迭代更新公式：
β 下一轮 = β 当前 - (二阶海森矩阵的逆)・一阶梯度向量

### 3.4 线性判别分析 LDA（Fisher 判别）
核心思想：投影后同类样本尽可能聚拢，异类样本中心距离尽可能大。
定义：μ0、μ1 为两类均值向量，Σ0、Σ1 为两类协方差矩阵
类内散度矩阵 Sw = Σ0 + Σ1
类间散度矩阵 Sb = (μ0 - μ1)・(μ0 - μ1) 的转置
优化目标广义瑞利商 J (w) = (w 转置 Sb w) / (w 转置 Sw w)
最优投影方向 w = Sw 的逆・(μ0 - μ1)
多分类 LDA：全局散度 St = Sw + Sb，求解 Sw 逆・Sb 的前 N-1 个广义特征向量，投影至 N-1 维空间。

### 3.5 多分类学习
核心思路：拆解多分类为若干二分类任务，三种主流拆分策略
OvR 一对其余：共 N 个分类器，每一类单独作为正类，其余全部为反类；预测时选择置信度最高类别。训练量大，测试开销小。
OvO 一对一：N 类两两配对，总共有 N*(N-1)/2 个二分类器；测试投票，票数最高为结果。训练集小，测试开销大。
ECOC 纠错输出码：M 个二分类器，每轮划分部分类别正负；训练编码、测试解码，对比海明距离预测，具备纠错能力。码长越长纠错效果越好，计算开销同步上升。

### 3.6 类别不平衡问题
训练集中正负样本数量差距极大时，直接使用 0.5 阈值会失效。
原始判别规则：y/(1-y) > 1 预测正例
平衡修正规则：y/(1-y) > m 正 /m 负，m 正、m 负分别为训练集正负样本数量
等价再缩放公式：新几率 = 原几率 × (m 负 /m 正)
三种解决手段：
欠采样：剔除多数类样本，平衡样本数量，代表 EasyEnsemble；
过采样：插值生成少数类新样本，代表 SMOTE；
阈值移动：训练不变，预测时直接调整判别阈值。
代价敏感学习可将 m 负 /m 正替换为误分类代价比值。
