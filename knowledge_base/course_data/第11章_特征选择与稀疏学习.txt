## 第 11 章  特征选择与稀疏学习

### 11.1 子集搜索与评价

### 11.1.1 特征选择核心意义
原始数据集往往包含大量冗余、无关、噪声特征：无关特征不携带任务判别信息，冗余特征与已有特征高度线性相关，二者都会提升模型训练开销、加剧过拟合、降低泛化性能。特征选择是从原始 d 维特征中挑选最优特征子集，不做特征变换，仅剔除无用维度，区别于 PCA 等降维（降维是特征线性组合生成新特征）。
特征选择完整流程分为两步：子集搜索、子集评价，二者组合构成一套特征选择算法。

### 11.1.2 子集搜索策略
特征子集空间规模随特征数量指数增长，d 个特征总子集数量为 2^d，暴力枚举全部子集计算量完全不可行，因此采用贪心启发式搜索，分为三类主流搜索方向：
前向搜索（自底向上）
初始子集为空集，迭代每轮向当前子集加入一个使评价指标最优的特征，直到新增特征无法提升指标，停止迭代。优点起步简单、迭代轮数少；缺陷容易陷入局部最优，早期选中无用特征后无法剔除。
后向搜索（自顶向下）
初始子集包含全部原始特征，每轮删除一个当前子集中最差特征，删除后指标不再下降则终止。适合特征数量较少场景，可一次性剔除冗余组合，计算开销大于前向搜索。
双向搜索（双向贪心）
前向增加特征与后向删除同步执行，每轮一边增优特征、一边删劣特征，兼顾两种策略优势，局部最优风险更低，但算法逻辑更复杂。
除贪心外还有遗传算法、模拟退火等全局搜索方法，能逼近全局最优，但迭代计算成本极高，工程极少使用。

### 11.1.3 子集评价标准
给定特征子集，衡量其区分能力的量化指标，和决策树划分指标同源：
信息增益：子集划分后数据集纯度提升越大，子集质量越高；
信息增益率、基尼指数均可作为评价依据；
本质逻辑：优质特征子集能够大幅降低样本不确定性，区分正负 / 不同类别样本。

### 11.2 过滤式选择 Filter
核心思想
特征筛选过程独立于后续学习器，先通过统计指标对所有特征打分，依据分数阈值直接剔除低分特征，筛选完成后再训练模型。筛选与训练解耦，速度极快，适合超高维稀疏大数据（文本、基因数据）。
经典算法：Relief（二分类专用）
算法流程
输入数据集、特征集，对每个样本 xi 执行两步近邻查找：
猜中近邻 xi,nh：和 xi 属于同一类别、距离最近样本；
猜错近邻 xi,nm：和 xi 属于不同类别、距离最近样本；
特征权重更新规则
对特征 j，若 xi 与 nh 在 j 上差距小、与 nm 差距大，说明特征区分能力强，权重增加；反之权重减少。遍历全部样本后取均值得到特征总权重，按权重从高到低排序，保留前 k 个特征。
拓展算法 Relief-F
适配多分类任务，每个样本除本类近邻外，遍历所有其他类别分别寻找猜错近邻，按各类样本数量加权计算权重，消除类别数量不平衡带来的偏差。
过滤式优缺点
优势：计算速度快、无训练迭代开销、可并行预处理、通用不依赖特定模型；
缺陷：仅使用统计单变量指标，不考虑特征组合协同作用，容易丢弃组合有效但单特征分数低的特征。

### 11.3 包裹式选择 Wrapper
核心思想
直接以目标学习器的泛化性能作为子集评价标准，将学习器嵌入特征搜索循环：每生成一个候选特征子集，训练模型并验证集精度作为得分，选出验证性能最优子集。特征筛选完全贴合当前模型，适配性远高于过滤式。
经典 LVW 算法（Las Vegas Wrapper）
流程：给定迭代上限，随机生成特征子集，训练模型并记录验证误差；
约束：仅当新子集误差低于当前最优时更新最优子集；
终止条件：达到最大迭代次数或连续多轮无更优子集；

优缺点
优势：充分考虑特征间协同关联，选出的子集在该学习器上效果最优；
致命缺陷：每一轮搜索都要完整训练、验证模型，计算开销极大，特征数稍多则无法落地，仅用于小规模数据集。

### 11.4 嵌入式选择与 L 正则化
嵌入式核心逻辑
特征选择与模型训练同步完成，在损失函数中加入特征权重惩罚项，训练过程自动将无关特征权重压缩至 0，等价完成特征剔除，兼顾过滤速度与包裹适配性。
L0、L1、L2 三种正则对比
L0 范数惩罚：约束权重中非零参数数量最小，完美实现特征选择；但 L0 优化是 NP 难问题，无法高效求解，仅理论存在；
L1 正则（LASSO）：L0 的凸松弛近似，损失函数增加 λ 乘以所有权重绝对值之和，训练时大量冗余特征权重会被压缩为 0，天然产生稀疏权重，自动筛选特征；
损失函数标准形式：Loss = 经验损失 + λ * 求和 | wj|
λ 越大，稀疏程度越强，保留特征越少；
L2 正则（岭回归）：惩罚权重平方和，仅缩小权重但不会置零，无法实现特征选择，仅缓解过拟合。
LASSO 求解特性
L1 损失不可导，无法直接梯度下降，常用坐标下降法、近端梯度下降迭代求解。多特征高度相关时，LASSO 会随机仅保留其中一个特征，存在不稳定性。
弹性网 Elastic Net
融合 L1 与 L2 惩罚项，解决 LASSO 在高度相关特征上的不稳定问题，同时实现稀疏选择、避免特征随机丢弃。

### 11.5 稀疏表示与字典学习
稀疏定义
向量 / 矩阵稀疏指绝大多数元素取值为 0，仅少量非零元素携带关键信息。稀疏表示目标：把原始稠密样本 x 转化为稀疏编码向量 z，x ≈ D・z，其中 D 为字典矩阵，z 仅有极少非零分量。
字典学习完整优化目标
最小化 全体样本重构误差 + 稀疏约束
约束：编码 z 的 L0 范数尽可能小，工程替换为 L1 范数松弛；
优化分两步交替迭代：
稀疏编码阶段：固定字典 D，对每个样本求解稀疏系数 z（匹配追踪 MP、基追踪 BP）；
字典更新阶段：固定所有稀疏编码 z，更新字典 D 最小重构误差；

应用场景
图像压缩、图像去噪、语音特征提取、高维生物数据降维；稀疏特征更容易线性可分，降低模型训练难度。

### 11.6 压缩感知
基础理论：奈奎斯特采样定理
传统信号采样要求采样频率≥2 倍信号最高频率，采样数据量大，存储传输成本高；压缩感知突破该限制。
核心前提
信号在某一组基下具备稀疏表示，可远低于奈奎斯特频率采样，通过少量观测值完整恢复原始信号。
两大核心条件
稀疏性：原始信号可稀疏表示；
有限等距性质（RIP）：观测矩阵满足约束，保证稀疏向量不会混淆；
求解逻辑
少量观测向量 y = Φ・x，Φ 为观测矩阵，x 稀疏，通过 L1 最小化重构原始完整信号；广泛用于医学 CT 成像、遥感信号、无线通信采样。

### 11.8 休息一会儿 小故事

LASSO 算法由 Robert Tibshirani 于 1996 年提出，最初用于统计学线性模型变量筛选，解决 L0 范数难以求解的难题，凭借 L1 天然稀疏特性成为嵌入式特征选择标杆。压缩感知理论 2006 年前后成型，颠覆传统采样规则，大幅降低医疗 CT 扫描辐射剂量，是理论落地医疗的经典机器学习成果。
